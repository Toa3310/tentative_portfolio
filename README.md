## ③ タスク管理アプリ（制作中）

### 概要
「今日の調子」に合わせて勉強内容を最適化する、コンディション連動型のタスク管理Webアプリ。
既存の学習管理ツールの「入力が面倒で続かない」「調子が悪いのに無理な計画を立てて挫折する」という課題を解決するために開発中。

### 使用技術
Python (Flask)
SQLite
HTML / CSS

### 実装予定の機能
* コンディション・チェック: 起動時に「集中度」と「体調」を10段階で入力。
* タスク自動提案: 入力された調子に合わせ、低負荷〜高負荷の最適なタスクテンプレートを動的に提案。
* 達成率・リズム可視化: 日付ごとのタスク達成率をグラフ表示し、特定の日付を選択すると、その日のコンディション詳細と実行タスクをドリルダウンで確認できるダッシュボード機能。

### 現在の進捗
* CLI版にて、コンディション入力とタスク提案の基本ロジックを検証中。
* Flaskを用いたWeb版への移行、およびSQLiteによるデータベース設計（ユーザー・タスク・コンディションデータの紐付け）に着手。

### この開発での挑戦・狙い
* 実用性とUXの追求: ユーザーの入力負荷を極限まで減らしつつ、データの「振り返り」の価値を最大化するUI設計。
* データベース（SQLite）の導入: データの永続化と、リレーショナルデータベースを用いた効率的なデータ管理の習得。
