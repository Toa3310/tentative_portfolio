## 概要

これまでに作成したアプリケーションをまとめています。
就活時に少しでもアピールになったり,自身のプログラミングスキルの向上を目的として取り組んでいます。

---

## ① シーザー暗号ツール

### 概要

シーザー暗号の暗号化・復号化を行うツール。
CLI版とWeb版の両方を実装。

### 使用技術

* Python
* Flask
* HTML

### 主な機能

* 暗号化 / 復号
* シフト数入力による変換
* エラーハンドリング（未入力・不正入力）

### 工夫した点

* 処理ごとに関数を分割し、可読性を向上
* CLI版のロジックをWeb版に再利用
* 入力値に対する例外処理を追加

### 学び

* 関数分割による設計の重要性
* Flaskを用いた基本的なWebアプリ開発
* GitHubでのバージョン管理
* 開発の基本的な流れ

### URL

* GitHubリポジトリ: https://github.com/Toa3310/caesar-cipher

## ② 学習記録アプリ

### 概要

日々の学習時間を記録・管理するWebアプリ。

### 使用技術

* Python
* Flask
* HTML / CSS

### 実装機能

* 学習時間の記録
* 日付ごとのデータ管理
* 記録の一覧表示

### 現在の進捗
CLI版: 完成済み
* csv保存・読み込み
* コメントを書けるように
* 一部の例外処理
* 平均学習時間の算出・表示

web: 完成済み(flaskで再構築)
* CLIの機能に加え,ルーティング設計も追加
* 記録を古い順に表示
* カード形式表示で視認性を向上
* テンプレート継承(base.html)

### 学び

- FlaskによるWebアプリの基本構造
- ルーティング設計（GET / POST）
- テンプレートエンジン（Jinja2）の活用
- CSVを用いたデータ管理
- 機能分割による設計の重要性
- Gitによるバージョン管理

* GitHubリポジトリ: https://github.com/Toa3310/study_record
