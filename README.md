## ① シーザー暗号ツール

### 概要

シーザー暗号の暗号化・復号化を行うツール。
CLI版とWeb版の両方を実装。

### 使用技術

* Python
* Flask
* HTML

### 主な機能

* 暗号化 / 復号
* シフト数入力による変換
* エラーハンドリング（未入力・不正入力）

### 工夫した点

* 処理ごとに関数を分割し、可読性を向上
* CLI版のロジックをWeb版に再利用
* 入力値に対する例外処理を追加

### 学び

* 関数分割による設計の重要性
* Flaskを用いた基本的なWebアプリ開発
* GitHubでのバージョン管理
* 開発の基本的な流れ
